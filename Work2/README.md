## 一、项目架构

本实验单独放在 `Work2` 文件夹中，主要文件如下：

```text
Work2/
├── README.md
├── main.py
└── assets/
    └── demo.gif
```

`main.py` 中实现了模型矩阵、视图矩阵、透视投影矩阵和 MVP 变换，并负责将三维三角形映射到屏幕坐标后绘制出来。

## 二、代码逻辑

程序先定义三角形的三个三维顶点，然后通过 `get_model_matrix()` 构造绕 Z 轴旋转的模型矩阵，通过 `get_view_matrix()` 构造相机视图矩阵，通过 `get_projection_matrix()` 构造透视投影矩阵。

每一帧中，程序按照 `projection @ view @ model` 的顺序计算 MVP 矩阵，将三角形顶点从模型空间变换到裁剪空间；随后执行透视除法，把齐次坐标转换到 NDC 坐标，再映射到窗口坐标进行线框绘制。

## 三、实现功能

- 实现绕 Z 轴旋转的模型变换。
- 实现视图变换与透视投影变换。
- 实现 MVP 矩阵组合与透视除法。
- 使用 Taichi GUI 绘制彩色线框三角形。
- 支持按 `A` / `D` 键控制三角形旋转。

简单运行方式：

```powershell
uv run python Work2/main.py
```
