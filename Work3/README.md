# 计算机图形学实验三：贝塞尔曲线

## 一、项目架构

本实验单独放在 `Work3` 文件夹中，主要文件如下：

```text
Work3/
├── README.md
├── main.py
└── assets/
    └── demo.gif
```

`main.py` 负责鼠标交互、De Casteljau 曲线计算、控制多边形绘制和曲线实时渲染。

## 二、代码逻辑

程序使用列表保存用户点击生成的控制点。若控制点数量不少于 2 个，就在 CPU 端使用 De Casteljau 算法对参数 `t` 从 0 到 1 进行采样，得到曲线上的一组点。

绘制时，程序先用灰线连接控制点形成控制多边形，再用绿色线段连接采样点形成贝塞尔曲线，最后用红色圆点标出控制点。按下 `C` 键会清空控制点并恢复空白画布。

## 三、实现功能

- 鼠标左键添加控制点。
- 实时显示控制多边形。
- 使用 De Casteljau 算法计算贝塞尔曲线。
- 实时绘制绿色贝塞尔曲线。
- 按 `C` 键清空画布。

简单运行方式：

```powershell
uv run python Work3/main.py
```
